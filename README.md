## ğŸ“‹ Quick Start

1. **Create a GitHub Account** (if you don't have one)

   - Go to [github.com](https://github.com)
   - Click "Sign up"
   - Follow the registration process

2. **Create a New Repository**

   - Click the "+" icon in the top right
   - Select "New repository"
   - Name it: `healthy-families` (or any name you prefer)
   - Make it Public
   - Click "Create repository"

3. **Upload Your Files**

   - On your repository page, click "uploading an existing file"
   - Drag and drop these files:
     - `index.html`
     - `styles.css`
     - `README.md`
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to your repository Settings
   - Scroll down to "Pages" in the left sidebar
   - Under "Source", select "main" branch
   - Click "Save"
   - Your site will be published at: `https://yourusername.github.io/healthy-families/`

### Adding Photos

To add your own photos to the website:

1. Take photos of your meal preparation and finished dishes
2. Upload photos to your GitHub repository (create an `images/` folder)
3. Edit the `index.html` file
4. Replace the placeholder sections with:
   ```html
   <img
     src="images/your-photo-name.jpg"
     alt="Description of photo"
     loading="lazy"
   />
   ```

## Customization

You can easily customize this website:

- **Colors**: Edit the color codes in `styles.css` (search for color values like `#2d5016`)
- **Content**: Update meal plans, recipes, and tips in `index.html`
- **Photos**: Add your own images by uploading them and updating the HTML
- **Contact Info**: Add your social media or contact information in the footer
