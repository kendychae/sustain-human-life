## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Recommended 1200px wide for optimal quality
- **Compression**: Use moderate compression to balance quality and file size
- **Orientation**:
  - About section: Portrait or square works best
  - Recipe cards: Landscape (3:2 ratio)
  - Pantry staples: Landscape or square

## Accessibility

All images should have descriptive alt text that explains what's in the photo. This helps:

- Screen reader users understand the content
- SEO and search engine visibility
- Users with slow internet connections
